# HCMA-history-matching
Code for surrogate-assisted reservoir history matching (HCMA)
# HCMA: Spatiotemporal Surrogate Model for Reservoir History Matching

This repository provides the implementation of the HCMA surrogate model proposed in the paper:

> *An Efficient Spatiotemporal Surrogate Model for Automatic Reservoir History Matching with Long-Horizon Production Dynamics*

## Environment
- Python >= 3.8
- PyTorch >= 1.12
- numpy>=1.21
- h5py>=3.7
- scikit-learn>=1.1
- CUDA (optional, for GPU acceleration)
